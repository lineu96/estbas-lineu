\begin{frame}{O que são dados?}
\protect\hypertarget{o-que-suxe3o-dados}{}
\begin{itemize}
\tightlist
\item
  Dados são conjuntos de valores.
\item
  Podem ser de diferentes fontes, tais como estudos e experimentos.
\item
  Podem conter variáveis de diferentes tipos.
\item
  Podem surgir em formatos estruturados e não estruturados.
\end{itemize}
\end{frame}

\begin{frame}{Conjunto de dados}
\protect\hypertarget{conjunto-de-dados}{}
\begin{itemize}
\item
  Em Estatística, em geral, lidamos com dados estruturados em um formato
  tabular.
\item
  Variáveis são as características observadas em cada um dos elementos
  que pertencem à amostra.
\item
  Na prática precisamos de uma forma para representar indivíduos e
  variáveis.
\end{itemize}
\end{frame}

\begin{frame}{Conjunto de dados}
\protect\hypertarget{conjunto-de-dados-1}{}
\begin{itemize}
\tightlist
\item
  Um conjunto de dados considerado arrumado é aquele em que:

  \begin{itemize}
  \tightlist
  \item
    Cada coluna representa uma variável.
  \item
    Cada linha representa uma observação.
  \item
    Cada célula representa o valor observado.
  \end{itemize}
\item
  O conjunto de dados completo e sem tratamentos é denominado conjunto
  de dados brutos.
\end{itemize}
\end{frame}

\begin{frame}{Tipos de variáveis}
\protect\hypertarget{tipos-de-variuxe1veis}{}
\begin{itemize}
\item
  Na prática, podemos coletar variáveis de diferentes tipos e naturezas.
\item
  Existem dois tipos (básicos) de variáveis:

  \begin{itemize}
  \tightlist
  \item
    Numéricas (quantitativas).
  \item
    Não numéricas (qualitativas).
  \end{itemize}
\item
  Cada uma das classes tem suas ramificações.
\end{itemize}
\end{frame}

\begin{frame}{Variáveis quantitativas}
\protect\hypertarget{variuxe1veis-quantitativas}{}
\begin{itemize}
\tightlist
\item
  Variáveis Quantitativas: assumem valores numéricos.

  \begin{itemize}
  \tightlist
  \item
    Discretas: características mensuráveis que podem assumir apenas um
    número finito ou infinito contável de valores e, assim, somente
    fazem sentido valores inteiros.
  \item
    Contínuas: características mensuráveis que assumem valores em uma
    escala contínua, isto é, na reta real.
  \end{itemize}
\end{itemize}
\end{frame}

\begin{frame}{Variáveis Qualitativas}
\protect\hypertarget{variuxe1veis-qualitativas}{}
\begin{itemize}
\tightlist
\item
  Variáveis Qualitativas: são as características definidas categorias,
  ou seja, representam uma classificação dos indivíduos e não uma
  característica numérica.

  \begin{itemize}
  \tightlist
  \item
    Nominais: não existe ordenação nem peso entre as categorias.
  \item
    Ordinais: existe uma ordenação entre as categorias.
  \end{itemize}
\end{itemize}
\end{frame}

\begin{frame}{Cuidados com variáveis}
\protect\hypertarget{cuidados-com-variuxe1veis}{}
\begin{itemize}
\item
  Existem particularidades na classificação de variáveis devido a
  situações como:

  \begin{itemize}
  \tightlist
  \item
    Discretização de variáveis contínuas.
  \item
    Limitações em instrumentos de mensuração.
  \item
    Utilização de quantidades numéricas para representação de variáveis
    categóricas
  \item
    Dentre outras.
  \end{itemize}
\item
  Deve-se sempre estar atento a este tipo de situação pois podem levar a
  implicações nas análises e consequentemente nos resultados.
\item
  Existem outros tipos de variáveis que ocorrem em situações
  particulares que requerem técnicas específicas de análise.
\end{itemize}
\end{frame}

\begin{frame}{Dados observacionais x dados experimentais}
\protect\hypertarget{dados-observacionais-x-dados-experimentais}{}
Dados observacionais

\begin{itemize}
\tightlist
\item
  Observação passiva da realidade.
\item
  Sem modificação das condições.
\end{itemize}

Dados experimentais

\begin{itemize}
\tightlist
\item
  Intervenção na realidade.
\item
  Condições controladas.
\item
  Observação dos efeitos das intervenções.
\end{itemize}
\end{frame}

\begin{frame}{No que devemos pensar antes de analisar nossos dados?}
\protect\hypertarget{no-que-devemos-pensar-antes-de-analisar-nossos-dados}{}
\begin{itemize}
\tightlist
\item
  O que estamos interessados em avaliar?
\item
  Quais são as variáveis de interesse?
\item
  Quais são as variáveis que queremos avaliar se influenciam a variável
  de interesse?
\item
  Quais são os métodos disponíveis para análise de variáveis deste tipo?
\item
  Quais os métodos disponíveis que permitem responder nossa pergunta de
  pesquisa?
\item
  Como coletar os dados?
\item
  Os dados são válidos?
\end{itemize}
\end{frame}
